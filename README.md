## 🛠️ Features

-   **Modular Agent Definitions**: Specialized protocols and patterns for different domains located in the `agents/` directory.
-   **Structured Execution**: Each identity follows a clear, phased protocol to ensure high-quality outcomes.
-   **Built-in Safety**: Integrated security and quality constraints.
-   **Production-Ready SaaS Foundation**: Complete with authentication, database schema, agent orchestration engine, and a conversion-optimized landing page.

## 🚀 Getting Started

1.  **Follow the [SETUP.md](SETUP.md)** guide to initialize your environment and deploy the foundation.
2.  **Refer to the [AGENTS.md](AGENTS.md)** file for instructions on how to work with the specialized AI identities.

## 📂 Directory Structure

-   `agents/`: Contains the detailed definitions and protocols for each AI identity.
-   `AGENTS.md`: The central hub for agent activation and global instructions.
-   `SETUP.md`: Comprehensive guide for setting up and deploying the AgentHub SaaS.
-   `app/`: Next.js application routes and pages.
-   `src/`: Core logic, including authentication, orchestration, and UI components.
-   `prisma/`: Database schema and migrations.
-   `scripts/`: Utility scripts, including the agent worker.
-   `docs/reports/`: Strategic documents, ADRs, and agent-generated reports.
-   `README.md`: This file.
